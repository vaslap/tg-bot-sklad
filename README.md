# Telegram-бот для управления складом

## Описание проекта

Telegram-бот предназначен для автоматизации операций по управлению складскими запасами. Бот позволяет проверять остатки товаров, резервировать товары под заказы клиентов, фиксировать приход и отгрузку товаров, проводить инвентаризацию и получать сводную информацию по складу.

Проект разработан в рамках выполнения расчетно-графической работы по дисциплине «Применение современных программных средств».

## Основные характеристики

- Платформа: Telegram
- Язык программирования: Python 3.12
- Библиотека для работы с Telegram API: aiogram 3.x
- Хранение данных: SQLite
- Система контроля версий: Git, GitHub

## Функциональные возможности

| Команда | Описание |
|---------|----------|
| `/start` | Вывод приветственного сообщения и списка команд |
| `/check <артикул>` | Проверка остатков товара |
| `/reserve <артикул> <количество>` | Резервирование товара под заказ |
| `/receipt <артикул> <количество>` | Фиксация прихода товара от поставщика |
| `/ship <артикул> <количество>` | Фиксация отгрузки товара покупателю |
| `/inventory <артикул> <количество>` | Проведение инвентаризации (сверка остатков) |
| `/dashboard` | Вывод сводной информации по складу |

## Установка и запуск

### Требования

- Установленный Python версии 3.10 или выше
- Аккаунт в Telegram
- Токен Telegram-бота, полученный от @BotFather

### Инструкция по установке

1. Клонировать репозиторий на локальную машину:


git clone https://github.com/vaslap/tg-bot-sklad.git
cd tg-bot-sklad
